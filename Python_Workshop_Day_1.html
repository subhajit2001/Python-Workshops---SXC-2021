Introduction to Python¶

Python is a general-purpose interpreted, interactive, object-oriented, and high-level programming language. It was created by Guido van Rossum during 1985- 1990.

Following are important characteristics of Python Programming −

  • It supports functional and structured programming methods as well as OOP.
  • It can be used as a scripting language or can be compiled to byte-code for building large applications.
  • It provides very high-level dynamic data types and supports dynamic type checking.
  • It supports automatic garbage collection.
  • It can be easily integrated with C, C++ and Java.

Lists¶

(1) Empty Lists are initialized using [] or list()
(2) The indexation in Lists start from 0 and the last element is of index n-1, where length of list is n.
(3) The len() function gives length of any sequence/iterable in python.
(4) The append method of the class list can be used to add elements to the list
(5) The extend methond extends a list by what is provided as an argument to the function
(6) The lists are mutable, which can be altered after creation.
(7) The lists also have negative indexation, where $-1$ corresponds to the last element, whereas $-n$ corresponds to the first element.

Sets¶

The Set Data Structure is very similar to what you study in maths. Every element appears only once in a set.
(1) Empty set can be created using set()
(2) A set can contain any number and type of elements, but it cannot have any mutable elements as its member.
(3) Sets are not indexable in Python.
(4) The set class have various methods such as add(), discard(), union(), intersection(), issubset(), issuperset(), isdisjoint(), remove(), symmetric_difference()
(5) Sets can also be created using {}, but during such an initialisation, there must be elements inside it.

For any two sets $A$ and $B$,
$A \cup B$ can be obtained by A.union(B) or A|B
$A \cap B$ can be obtained by A.intersection(B) or A&B
$A \Delta B = (A-B) \cup (B-A)$ can be obtained by A.symmetric_difference(B) or (A-B)|(B-A)
Whether $A \subseteq B$ can be checked using A.issubset(B)
Whether $A \supseteq B$ can be checked using A.issuperset(B)
Whether $A \cap B = \phi$ can be checked using A.isdisjoint(B)

Functions in Python¶
(1) Functions in Python are defined using the def() keyword. 

(2) A function may or may not always return something.

(3) Sometimes, some additional data are required for the function to execute successfully. These are known as parameters, defined during function definition.

(4) When values are passed during a function call, these additional information are called arguments.

(5) Some types of arguments are:

(a) positional arguments: arguments passed during function call are mapped to the parameters based on their position

(b) keyword arguments: arguments, where we suppy the name of the parameter corresponding to the argument as it is passed into the function

(c) default arguments: some parameters have default values set to them during function definition, if no arguments are passed during function call, then also the function carries on with its work

(6) Non-default arguments and keyword arguments must preceede positional arguments.

def is_prime(n):
  if n==1:

  else:
    flag = 0
    for i in range(2 , (n//2)+1):
      if n%i==0:
        flag=1
        break
    if flag==1:
      return False
    else:
      return True

Population Variance of a list of numbers $x_1,x_2, \ldots x_n$ is given by 
$$\sigma^2 = \dfrac{1}{n} \sum_{i=1}^n(x_i-\overline{x})^2$$
The Sample variance is given by $$s^2 = \dfrac{1}{n-1}\sum_{i=1}^n(x_i-\overline{x})^2$$













In [119]:



    


def variance(x , ddof = 0):
  if len(x)==0 or len(x)==1:
    return 0
  s = 0
  for i in x:
    s = s+i
  avg = s/len(x)
  squared_deviations = [(i-avg)**2 for i in x]
  if ddof==0:
    variance = sum(squared_deviations)/len(x)
  elif ddof==1:
    variance = sum(squared_deviations)/(len(x)-1)
  return variance

# Here's a code for matrix multiplication.

def matmul(A,B):
  if len(A[0])==len(B):
    product = []
    for i in range(len(A)):
      product.append([])
      for j in range(len(B[0])):
        product[i].append(0)
    for i in range(len(A)):
      for j in range(len(B[0])):
        for k in range(len(A[0])):
          product[i][j]+=A[i][k]*B[k][j]
    return product
  else:
    print('Matrices A and B are not compatible for multiplication.')
    return -1
