Introduction to Numpy¶

Numpy, or Numerical Python is a python package for scientific computing, that finds wide usage in almost any application involving math and scientific stuff. We are going to discuss what are numpy arrays, how to create them, how they behave, and we will then perform some basic arithmetic with them.

Then for some slightly advanced topics, we will find the inverse of a matrix, and find the length of a vector. So lets get started!

In maths, we generally add two matrices pretty easily.
